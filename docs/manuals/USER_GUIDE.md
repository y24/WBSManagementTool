# WBS管理ツール 利用ガイド

本ツールは、プロジェクト、タスク、サブタスクの3階層で工程を管理し、ガントチャートで進捗を可視化するためのWebアプリケーションです。Excelのような操作感で、効率的なスケジュール管理と進捗把握を支援します。

---

## 1. 画面構成

画面は大きく分けて3つのエリアで構成されています。

1.  **フィルターパネル（上部）**: 表示対象の絞り込み、表示オプションの切り替え、Excel出力などを行います。
2.  **WBSツリー（左側）**: プロジェクトの構造をツリー形式で表示し、データの入力・編集を行います。
3.  **ガントチャート（右側）**: スケジュールをタイムライン形式で可視化します。ツリーと連動してスクロールします。

---

## 2. WBS管理（データの編集・操作）

### 2.1 データの階層構造
データは以下の3階層で管理されます。
- **プロジェクト**: 最上位の管理単位。
- **タスク**: プロジェクトを構成する大きな工程。
- **サブタスク**: 最小の作業単位。進捗や工数はここで管理します。

### 2.2 データの入力・編集
- **インライン編集**: 名称、ステータス、担当者、日付、進捗率、工数などは、表上のセルを直接クリックまたは選択することで編集できます。入力した内容は即座にサーバーへ保存されます。
- **詳細モーダル**: 行の右端などにある「詳細編集」ボタンから、メモやチケットID、詳細な説明を編集できます。
  - **チケットID**: 設定されたテンプレートに基づき、Azure DevOpsなどの外部システムへリンクされます。
- **日付のバリデーション**: 開始日が終了日を上回るような入力はエラーとなり、保存されません。

### 2.3 項目の追加・削除
- **追加**: 画面上部の「プロジェクト追加」ボタン、または各行の右側にある「＋」ボタンから、タスクやサブタスクを追加できます。
- **削除**: 項目を「Removed」ステータスに変更するか、または後述の一括操作で削除（論理削除）できます。削除時には、配下の項目を含めた影響範囲の数が表示されます。

### 2.4 並び替え
- 各行の左端をドラッグ＆ドロップすることで、同一階層内での並び替えが可能です。

### 2.5 一括操作（フローティングメニュー）
行の左側にあるチェックボックスを選択すると、画面下部に青いメニューが表示されます。
- **削除**: 選択した項目（およびその配下すべて）を削除します。
- **複製**: 計画の再利用やリプランのために、選択した項目をコピーします。
- **実績クリア**: 選択した項目の実績日や進捗率を一括でリセットします。
- **日付移動**: 全体のスケジュールを維持したまま、開始日を起点として一括で前後にずらします。

## 3. ガントチャートの活用

### 3.1 予定と実績の表示
スケジュールの状態に応じて、ガントチャート上のバーの表示が変化します。
- **計画バー（灰色）**: 予定期間を表示します。実績が入力されると補足的な細いバーへと切り替わります。右端の濃い部分は「レビュー期間」を表します。
- **実績バー（ステータスカラー）**: 実際に着手した期間を表示する太いバーです。
  - バーの色はステータス（進行中、完了など）に基づきます。
  - 進捗率を表示する設定にしている場合、実績バーの中央にパーセンテージが表示されます。

### 3.2 ドラッグ＆ドロップによる期間編集
ガントチャート上のバーを直接マウスで操作して、スケジュールを直感的に編集できます。
- **期間の移動**: バーの中央付近をドラッグすると、期間の長さを維持したまま開始日と終了日を平行移動できます。
- **期間の伸縮**: バーの左端または右端をドラッグすると、開始日または終了日のみを変更して期間を調整できます。
- **レビュー期間の調整**: 計画バーの右側にある濃い色のエリア（レビュー期間）の境界をドラッグすることで、レビュー開始日やレビュー期間を個別に変更できます。

### 3.3 ダブルクリックによる計画入力
計画（開始日・終了日）がまだ入力されていない項目に対して、素早く入力を開始できます。
- **クイック入力**: ガントチャートの空いている行をダブルクリックすると、その日付を起点とした1日分の計画が即座に作成されます。

### 3.4 マイルストーン（マーカー）
特定の重要な日付を「マーカー」として記録できます。
- **追加/編集**: ガントチャート上部の「日付ヘッダー」をクリックすると、マーカー編集モーダルが開きます。
- **表示**: 設定したマーカーはチャートを貫く垂直線と、ヘッダー上のラベルとして表示されます。

### 3.5 遅延検知
以下の条件に該当する場合、バーの右側に **⚠（警告アイコン）** が表示されます。
- 終了予定日を過ぎているのにステータスが「Done」でない。
- レビュー期間を含めた期限内にレビューが開始されていない。

## 4. フィルターと表示設定

### 4.1 フィルタリング
フィルターパネルから以下の条件で表示を絞り込めます。
- プロジェクト、ステータス、担当者、サブタスク種別の複数選択。
- **検索**: 名称によるキーワード検索。
- **遅延タスク**: 遅延が発生している項目のみを抽出します。
- **リセット**: すべてのフィルターを解除して全表示に戻します。

### 4.2 表示オプション（設定ボタン ⚙️）
画面右上のギアアイコンから、好みに合わせて表示を微調整できます。
- **完了済/Removedの表示切替**: 終了したプロジェクトやRemovedの項目を表示するかどうか。
- **今日の日付を強調**: 本日の日付を黄色の縦線でハイライトします。
- **進捗率・担当者の表示**: ガントバー付近に情報を表示するかどうか。
- **ダークモード**: 画面配色を暗色系に切り替えます。
- **表示オプションの保存**: これらの設定はブラウザに保存され、次回アクセス時も維持されます。

## 5. 便利な機能・Tips

### 5.1 Excel連携
- **エクスポート**: フィルターパネルのダウンロードボタンをクリックすると、**現在表示されている（フィルター適用済みの）データ**をExcel形式でダウンロードできます。

### 5.2 画面の調整
- **列幅の調整**: WBSツリーの「名称」列の境界線をドラッグすることで幅を変更できます。
- **表示レベルの切替**: ツリー上部のアイコン（リスト型、レイヤー型）から、全展開・プロジェクト単位・タスク単位の表示をワンクリックで切り替えられます。

### 5.3 キー操作
- **Escapeキー**: 編集中の詳細モーダルや、表示設定などのポップアップを閉じることができます。
- **未保存の確認**: 詳細編集モーダルで変更がある状態で閉じようとすると、破棄の確認メッセージが表示されます。
