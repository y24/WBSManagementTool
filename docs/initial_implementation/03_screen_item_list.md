# Web版WBS管理ツール 画面項目一覧

## 1. 前提

### 1.1 対象画面

本資料で対象とする画面は以下。

* メイン画面

  * ヘッダー領域
  * スライサー領域
  * WBSツリー領域
  * ガントチャート領域
* サブタスク詳細ダイアログ
* マスタ管理画面
* 共通ダイアログ

### 1.2 入力・保存の基本方針

* メイン画面のインライン編集項目は、原則として**自動保存**
* 詳細ダイアログ内の項目は、初版では**明示保存**を基本とする
* マスタ管理画面は、**明示保存**
* 削除やRemoved変更などの破壊的操作は確認ダイアログを挟む

# 11. 補足ルール

## 11.1 省略表示ルール

* 名称、担当者名などの長い文字列は1行省略表示とする
* ホバー時に全文をツールチップで表示する
* 編集時は省略解除する

## 11.2 日付表示ルール

* 通常表示は MM/DD
* ホバー時・編集時は YYYY/MM/DD
* バリデーションは完全日付で行う

## 11.3 保存ルール

* メイン画面の常時表示列は自動保存
* 詳細ダイアログとマスタ画面は明示保存
* 保存失敗時はユーザーに認識できること

## 11.4 ガント編集ルール

* 初版ではガントチャート上での直接編集なし
* ツリー側編集結果をリアルタイム反映する
